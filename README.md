# Puzzle
Puzzle Game

## Build Instructions

*Note - The project currently supports only Windows OS.*

### 1. Windows
#### Initial Requirements
1. C/C++ Compiler (VC++, MinGW_w64, clang, etc.)

2. Cmake

#### Building

1. ```git clone https://github.com/rohansiddhu/Puzzle```

2. ```cd Puzzle```

3. ```mkdir build```

4. ```cd build```

5. ```cmake ..```

6. ```cmake --build .```
